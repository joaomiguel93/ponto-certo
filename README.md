# ⏱ PontoCerto DP

**Ferramenta gratuita de apuração de cartão ponto para profissionais de Departamento Pessoal.**

Desenvolvida para contadores e analistas de DP que precisam apurar horas de forma rápida, sem sistemas complexos e sem custo.

🔗 **[Acessar agora →](https://joaomiguel93.github.io/dp-ponto-certo)**

---

## ✅ Funcionalidades

- Apuração mensal com todos os dias do mês gerados automaticamente
- Cálculo de **horas extras 50% e 100%** conforme CLT
- **HE 100% automático** em domingos e feriados
- Controle de **atrasos** com tolerância configurável (art. 58 CLT)
- **Adicional noturno** (22h–05h) com hora reduzida opcional (art. 73 CLT)
- Verificação de **intrajornada** com alerta de violação (art. 71 CLT)
- Suporte a **sábado trabalhado** com horário específico
- **Feriados nacionais** pré-cadastrados até 2028 + cadastro de municipais/estaduais
- Ocorrências por dia: **Falta, Atestado, Feriado**
- **Resumo mensal** com totais para folha de pagamento, incluindo datas de faltas e atestados
- **Múltiplos funcionários** com troca rápida
- **Salvamento automático** no navegador (localStorage)
- Exportação em **CSV** e impressão em **PDF** (paisagem)
- **Atalhos de teclado**: `F` Falta · `A` Atestado · `E` Feriado · `R` Limpar ocorrência
- Modo claro e escuro
- **100% offline** — roda direto no navegador, sem cadastro, sem servidor

---

## 🚀 Como usar

1. Acesse o link acima ou baixe o arquivo `ponto-certo-v5.html`
2. Abra no Chrome, Edge ou Firefox
3. Preencha os dados do funcionário e empresa
4. Configure a jornada (entrada, saída, intervalo)
5. Clique em **▶ Gerar Tabela**
6. Lance os horários diários ou use **📋 Preencher Padrão**
7. Exporte com **📥 CSV** ou imprima com **🖨️ PDF**

---

## ⚙️ Parâmetros configuráveis

| Parâmetro | Padrão |
|-----------|--------|
| Jornada diária | 08:00 – 17:00 (com intervalo 12:00–13:00) |
| Tolerância de atraso | 10 minutos |
| Sábado trabalhado | Não (configurável) |
| HE personalizada | Não (50%/100% padrão CLT) |
| Adicional noturno | Sim |
| Hora reduzida noturna | Sim (52min30s = 1h) |
